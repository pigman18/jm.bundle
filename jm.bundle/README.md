# JM 插件 for VS Code

> 一个面向 JM 漫画的 **本地搜索与下载辅助插件**  
> ⚠️ **第三方个人项目，仅供学习与研究使用，严禁用于任何商业或非法用途**

---

## 🤔 为什么做这个插件

在长期使用网页版的过程中，总会遇到一些琐碎但影响体验的问题：

- 下载任务偶尔失败，便被遗忘在角落，时间一长也就不再想起去完成它；
- 下载到本地的漫画只剩下编号，难以还原作品名称；
- 想要根据特定 tag 筛选符合口味的漫画时，本地资源又几乎无从下手。

为了缓解这些碎片化痛点，这个插件将**下载、本地搜索与灵活配置**整合进 VS Code，让漫画检索与管理更加直观、高效。

---

## ✨ 功能简介

- ⬇️ **便捷下载**：支持章节级批量下载，大幅降低中断风险
- 🤫 **静默运行**：用户无需感知具体操作过程，只管看漫画就行
- 📚 **本地搜索**：一键检索本地漫画资源，快速定位目标作品
- ⚙️ **灵活配置**：可自定义代理、下载目录等参数
- 🧩 **轻量集成**：以 VS Code 侧边栏形式存在，随时调用。**只要界面开着，就是在写代码，谁看谁迷糊**

## 📦 安装方式

### 从 VSIX 安装
1. 获取 `vscode-jm-bundle-x.x.x.vsix`
2. 打开 VS Code → `Extensions` → `…` → `Install from VSIX…`
3. 选择文件并确认安装

---

## 🚀 使用说明

1. 启动 VS Code
2. 切换到 **JM 插件** 侧边栏
3. 按需配置
4. 开始使用

尽情地享受吧

---

## 📁 项目结构

- **jm.bundle**：主模块，负责存储、爬虫、消息推送等核心逻辑
- **jm.bundle.front**：前端展示页面，包含基础查询、搜索与下载列表
- **jm.bundle.vscode**：VS Code 插件层，负责将主模块嵌入 VS Code 侧边栏
