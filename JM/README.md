## 架构说明

### 主机发现机制

API 域名**不硬编码**，启动时从以下 CDN 地址获取 `apiUrl`：

| 优先级 | URL |
|--------|-----|
| 主 | `https://rup4a04-c02.tos-cn-hongkong.bytepluses.com/newsvr-2025.txt` |
| 备用 | `https://rup4a04-c01.tos-ap-southeast-1.bytepluses.com/newsvr-2025.txt` |

获取到的文本内容经解析后，`apiUrl` 存储到 `localStorage["apiUrl"]` 中。

### 接口地址格式

```
GET  {apiUrl}/{path}?param1=value1&param2=value2
POST {apiUrl}/{path}
```

### 通用响应格式

```json
{
  "code": 200,
  "data": { ... },
  "msg": "..."
}
```

- `code === 200` 表示请求成功
- 部分接口的 `data` 层还会包含 `status: "ok" | "error"` 和 `msg`

## 通用请求头 & 鉴权

每次 HTTP 请求自动附加以下 Headers：

| Header | 值 | 说明 |
|--------|-----|------|
| `Tokenparam` | `{unix_timestamp},{version}` | 例如 `1718198400,2.0.25` |
| `Token` | `AES-ECB({timestamp}185Hcomic3PAPP7R)` | 时间戳 + 固定密钥 `185Hcomic3PAPP7R` 的 AES-ECB/ZeroPadding 加密值 |
| `Authorization` | `Bearer {jwttoken}` | 从 `localStorage["jwttoken"]` 读取 |
| `Cookie` | `AVS={memberInfo.s}` | 从 `localStorage["memberInfo"].s` 读取 |

### Token 生成逻辑

```
version = "2.0.25"
timestamp = Math.floor(Date.now() / 1000)
Tokenparam = timestamp + "," + version
raw = String(timestamp) + "185Hcomic3PAPP7R"
Token = AES_ECB_encrypt(raw, key="185Hcomic3PAPP7R")
```

#### 获取 App 设置 & 主机信息

| 项 | 值 |
|----|-----|
| 路径 | `{apiUrl}/setting` |
| 方法 | GET |
| 用途 | 获取应用配置（包括 `main_web_host`），用于初始化 |

## LocalStorage 存储键说明

| Key | 用途 |
|-----|------|
| `apiUrl` | API 基础 URL（应用启动时动态获取） |
| `hostServer` | 主机服务器列表 |
| `main_web_host` | 分享链接使用的主机名 |
| `jwttoken` | JWT 鉴权 Token |
| `memberInfo` | 用户信息（JSON，含 `uid`, `s`(Cookie) 等） |
| `memberAccount` | 用户账号信息（用于自动登录） |
| `imageSource` | 图片来源选择（`"0"`/`"1"`） |
| `darkMode` | 深色模式开关 |
| `lang` | 语言设置 |
| `adsList` | 广告列表缓存 |
| `adsContent` | 广告内容缓存 |
| `likedItems` | 已点赞项目列表 |
| `novelLikeItems` | 小说点赞列表 |
| `novelMarkItems` | 小说收藏列表 |
| `novelFontSizeClass` | 小说字体大小设置 |
| `novelLang` | 小说语言设置 |
| `novelRead` | 小说阅读进度 |
| `trackList` | 追踪列表 |
| `search_history` | 搜索历史 |
| `addedMarks` | 已添加收藏标记 |
| `removedMarks` | 已取消收藏标记 |
